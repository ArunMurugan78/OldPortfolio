Projects

Blogzz

Blogzz is a blogging web application. Users can sign up and create posts.It has Features to like , bookmark and search for a blog and also all the CRUD operations on the blog. Users can also follow another user . The user gets a notification if some one likes their post ,starts following him/her and also when a user followed by them posts a blog. It is made using React.js for frontend and Express for backend . Other tools/technologies/libraries used include bootstrap , mongoDB , passport.js for authentication , fuse.js for searching , Helmet.js for protection , Mongoose . It is deployed on heroku .
